# Adaptive Wildfire Sensing Research Tool (AWSRT)

Research-first tool with Environment/Fire/Sensors designers, Run, Replay/PT, and Analysis.
Back end: FastAPI (Python). Front end: Next.js (React). Logs: Parquet; fields: Zarr.

## Dev quickstart
- Backend: uvicorn api.main:app --reload --port 8000
- Frontend: (Next.js) npm run dev (or pnpm dev) at http://localhost:3000
